Collaborative

   This example shows how to build a collaborative graph for multiple
   users with React Flow and [76]yjs. The changes made in this graph are
   visible for all users on this page. To see it working, you can [77]open
   this page in another tab or on another device or browser. The example
   can be used as a starting point to build multi user flows and editors.

   This is a Pro example. Get [78]all pro examples, 1:1 support from the
   xyflow team and prioritized Github issues with a React Flow Pro
   subscription.
