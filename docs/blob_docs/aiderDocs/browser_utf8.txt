Aider in your browser

   [84]Aider browser UI demo video

   Use aider's new experimental browser UI to collaborate with LLMs to
   edit code in your local git repo. Aider will directly edit the code in
   your local source files, and [85]git commit the changes with sensible
   commit messages. You can start a new project or work with an existing
   git repo. Aider works well with GPT 3.5, GPT-4, GPT-4 Turbo with
   Vision, and Claude 3 Opus. It also supports [86]connecting to almost
   any LLM.

   Use the --browser switch to launch the browser version of aider:
python -m pip install -U aider-chat

export OPENAI_API_KEY=<key> # Mac/Linux
setx   OPENAI_API_KEY <key> # Windows, restart shell after setx

aider --browser
